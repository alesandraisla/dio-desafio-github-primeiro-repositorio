Em coleções chaveadas é possível utilizar o loop for..of

## Map ## {{Coleção chaveada}}
Permite chaves de qualquer tipo
Coleção de arrays no formato [chave, valor].
pode ser iterado por um loop for...of.
Podem ter chaves de qualquer tipo
Possui propriedade length .
São faceis de iterar.
Utilizado quando o valor das chaves é desconhecido.
Os valores tem o mesmo tipo. 

## Object ## {{Não são}}
Não permite chave de qualquer tipo 
Tem chaves no formato de string
Precisa iterar nas propriedades para saber o tamanho do objeto
Utiliza quando conhece o valor da chave
Não necessariamente precisa ter valores do mesmo tipo.

## Set ##  {{Coleção chaveada}}

São estruturas que armazenam apenas valores únicos que não se repetem.
Em vez da propriedade length, para retornar o tamanho consulta-se o número de registros pela propriedade size.
Não possui os métodos map, filter, reduce, etc de um array. 

## Array ## {{Não são}}
É mais flexível, pois tem mais métodos.
